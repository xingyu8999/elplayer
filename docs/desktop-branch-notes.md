# 桌面版分支说明

## 目标

桌面版分支用于把 Web 播放器扩展成桌面歌词软件。重点能力包括：

- 透明桌面歌词浮窗
- 极简黑胶机浮窗
- 窗口置顶
- 鼠标穿透
- 系统托盘
- 全局快捷键
- 桌面设置面板

## v5.3.4 变化

- 修复窗口化时底部播放器过大和 UI 重叠问题。
- 修复极简黑胶机小尺寸下文字显示不全问题。
- 中文优先整理发布物料和公告。
- 准备官网展示页、下载页和 macOS 签名/公证说明。

## 不在本分支内处理的内容

- React / Next 重构
- 移动端原生 App
- 在线音乐版权源
- 云同步账号系统

## English Summary

The desktop branch adds transparent overlay, mini vinyl mode, tray controls, global shortcuts, and compact windowed layout improvements.
