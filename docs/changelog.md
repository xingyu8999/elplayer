# 更新日志

## v5.3.4

本版本为真实发布前修正版，重点围绕中文发布物料、窗口化布局、极简黑胶机小尺寸和官网发布准备。

### 已修复

- 将 README、Release 模板、下载页、官网展示页和主要发布说明改为中文优先；需要英文时放在中文之后。
- 修复 Electron 窗口化时底部播放器过大、控制区堆叠、文字显示不全的问题。
- 新增窗口化紧凑布局：小窗口下隐藏音量条、倍速、歌词微调、透明歌词等次要按钮，保留上一曲、播放/暂停、下一曲等必备控制。
- 修复桌面浮窗“极简尺寸：小”时 UI 和文字显示不全的问题。
- 极简小尺寸下优先显示黑胶动效、歌名和歌手；上一曲/下一曲等控制按钮改为悬停显示。

### 已新增

- 新增 `website.html`，可作为官网展示页或二级页面使用。
- 更新 `download.html`，按 GitHub Release v5.3.4 的正式附件路径预置下载链接。
- 新增中文优先的 GitHub Release 模板。
- 新增发布前验收清单，区分“已准备文件”和“需要账号/证书才能执行的动作”。

### 需要用户在本机完成

- 上传 GitHub Release 附件。
- 将 `download.html` 中的下载链接确认到真实可访问状态。
- 在 macOS 设备上使用 Developer ID 证书完成签名和 Apple 公证。
