# 桌面端优化说明

## v5.3.4 重点

本版本重点处理窗口化和极简小尺寸问题：

- 主窗口缩小时，底部播放器自动进入紧凑布局。
- 小窗口下隐藏音量条、倍速、歌词微调、透明歌词等次要按钮。
- 保留上一曲、播放/暂停、下一曲等必备控制。
- 极简黑胶机“小”尺寸优先保证歌名、歌手和黑胶动效显示。
- 极简小尺寸下上一曲/下一曲按钮改为悬停显示，减少横向挤压。

## 为什么这样设计

桌面歌词类软件的窗口经常处于小尺寸或贴边状态。如果控件过多，容易出现遮挡、换行、文字截断和按钮重叠。因此窗口化场景应遵循：

```text
优先显示正在播放内容
优先保留播放控制
隐藏低频设置项
把复杂设置放回设置面板
```

## 后续优化方向

- 增加“紧凑播放器 / 标准播放器”手动开关。
- 允许用户自定义底部播放器按钮。
- 给极简黑胶机增加仅图标模式。
- 增加窗口吸附边缘功能。

## English Summary

v5.3.4 improves compact windowed layout and mini vinyl readability. Secondary controls are hidden in small windows to preserve essential playback actions.
