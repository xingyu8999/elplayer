# 竞品分析与产品优化说明

## 产品定位

Elplayer 不应只定位为音乐播放器，而应定位为：

```text
本地媒体 + 桌面歌词 + 视频字幕 + 极简播放控制 + 网站嵌入
```

## 竞品启发

### 桌面歌词类

优点：轻、悬浮、跟随工作流。  
缺点：通常只解决歌词显示，媒体能力弱。

### 移动歌词类

优点：全屏歌词、沉浸感强、触控体验成熟。  
缺点：对本地视频、字幕和桌面浮窗场景支持有限。

### 综合播放器类

优点：播放能力完整。  
缺点：界面偏重，不适合长期悬浮和轻量使用。

## Elplayer 的优化方向

- 保持 Web 版轻量。
- 桌面端强化透明浮窗、极简模式、置顶和鼠标穿透。
- 移动端强化全屏歌词、视频字幕和触控布局。
- 发布物料统一中文优先，后续按需要补英文。

## v5.3.4 针对性优化

- 小窗口下减少控制按钮，避免重叠。
- 极简黑胶机小尺寸优先保证歌名和歌手显示。
- 官网、下载页、Release 模板均改成中文优先。

## English Summary

Elplayer differentiates itself by combining local media, desktop lyrics overlay, video subtitles, mini player controls, and website embedding.
