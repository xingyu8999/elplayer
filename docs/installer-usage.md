# 安装包使用说明

> 本说明先写中文；英文只在必要时作为补充。

## Windows 安装

1. 从 GitHub Release 或官网下载页下载 x64 安装包。
2. 双击运行：

```text
Elplayer-5.3.4-Setup-x64.exe
```

3. 选择安装目录。
4. 安装完成后检查桌面快捷方式、开始菜单和托盘图标。

## Windows 安装后检查

- 主窗口可以打开。
- 桌面歌词浮窗可以显示 / 隐藏。
- 极简黑胶机模式可以切换。
- 鼠标穿透开启后可以通过主窗口、托盘或快捷键恢复。
- 右下角托盘显示 Elplayer 图标。

## macOS 安装

1. 下载与你设备匹配的 DMG 或 ZIP。
2. 将 Elplayer 拖入 Applications。
3. 首次启动如遇安全提醒，请优先使用已签名和公证版本。

## macOS 发布要求

公开分发前建议完成：

- Developer ID 签名
- Apple 公证
- stapler 附加公证票据

详见：`docs/macos-signing-notarization.md`

## 常见问题

### 鼠标穿透后点不到浮窗怎么办？

使用以下任一方式恢复：

- 主窗口右上角“桌面设置”
- 系统托盘菜单
- 快捷键 `Ctrl / Cmd + Shift + P`

### 窗口化后底部播放器太大怎么办？

v5.3.4 已加入窗口化紧凑布局。小窗口下会自动隐藏部分次要按钮，只保留播放必备控制。

### 极简黑胶机小尺寸文字不完整怎么办？

v5.3.4 已修复。小尺寸会优先显示黑胶动效、歌名和歌手；上一曲/下一曲按钮改为悬停显示。
