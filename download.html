安装前说明

  • Windows 默认发布 x64 安装包，不再默认构建 ia32。
  • macOS 对外分发前需要完成 Developer ID 签名与 Apple 公证。
  • MKV / AVI 播放取决于系统和 Chromium 编码支持，推荐 MP4 / H.264 / AAC。
  • 鼠标穿透开启后，可以从主窗口“桌面设置”、托盘菜单或快捷键恢复控制。
